# EMU-SIC Web

**Simulador Educativo del Procesador SIC (Small Instructional Computer)**

Migración moderna a la web del proyecto original EMU-SIC. Permite la exploración, programación y ejecución ciclo a ciclo de la arquitectura SIC directamente desde el navegador.

## Características
- **FSM Fiel:** Emulación ciclo a ciclo de la Unidad de Control (~50 estados).
- **Ensamblador Integrado:** Soporte completo para directivas `ORG`, `DATA`, `SYM` y MRI/IOP.
- **Salida de Video:** Mapeo de memoria directo (40x25) con soporte de atributos (Color, Brillo, Parpadeo).
- **Modo No-Delay:** Ejecución de alta velocidad para programas complejos.
- **Documentación:** Manuales técnicos y de usuario integrados en PDF.

## Stack Tecnológico
- HTML5 / CSS3 (Dark Mode)
- JavaScript Vanilla (ES6+)
- TypedArrays para emulación de hardware de alta eficiencia.
